전달 사항

📌 펌웨어 제작 요청 시 공통 전달 조건

  • 샘플 번호별로 독립된 메인보드(또는 펌웨어 버전)으로 제작 요청 — 동일 보드 내 코스 선택 방식 불가
  • 건조 타임은 기존 코스값 유지 (변경 없음) — 본 표는 세척+헹굼 구간만 해당
  • 수조통 용량: 4,500cc 기준 (현 3,800cc → 개선 예정) — 실제 샘플 제작 시 개선된 수조통 적용 요청
  • 각 공정회차 종료 시 완전 배수 후 다음 회차 진입 필수 (기존 Normal/Glass 배수 미발생 버그 수정 전제)
  • 공정당 물량 최소 850cc 이상 필수 / 권장 900cc — 미만 시 배관 충진 불량 가능성
  • 세척(1회차)는 배관 에어 퍼지 고려 → 항상 해당 샘플 내 최다 물량 유지 요청
  • 수령 후 실측 항목: 실제 배수량, 세척 성능(잔류물), 공정별 시간 측정 진행 예정
