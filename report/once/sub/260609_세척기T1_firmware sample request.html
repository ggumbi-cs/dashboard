📌 펌웨어 제작 요청 시 공통 전달 조건

  • 샘플 1개 = 강력·표준·급속 3코스 세트 — 총 5샘플 × 3코스 = 펌웨어 15종 독립 제작 요청
  • 동일 샘플 내 3코스는 물량 구성 동일, 세척·헹굼 시간만 상이
  • 건조 타임은 기존 코스값 유지 — 본 표는 세척+헹굼 구간만 해당
  • 수조통 용량 4,500cc 기준 — 개선된 수조통 적용 요청
  • 각 공정회차 종료 시 완전 배수 후 다음 회차 진입 필수 (기존 배수 미발생 버그 수정 전제)
  • 공정당 물량 최소 850cc 이상 필수 / 권장 900cc
  • 수령 후 실측 항목: 배수량, 세척 성능(잔류물), 공정별 시간 측정 예정
