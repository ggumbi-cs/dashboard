· 나도 할 수 있다
POINT 01
🚀
처음부터 완벽할 필요 없다
가볍게 시작해도 됩니다. 대화가 쌓이면서 툴이 발전합니다.
POINT 02
💡
AI가 먼저 기능을 제안한다
내가 생각 못한 기능도 AI가 제안해줍니다. 제안을 받아들이면 됩니다.
POINT 03
🔄
마음에 안 들면 다시 말하면 된다
수정 요청도 대화로 하면 됩니다.
"전체 확대 말고, 마우스 올린 PDF만 확대되게 해줘."
POINT 04
🛠
대화가 곧 개발 과정이다
요청 → 결과 → 수정 → 개선, 이 반복이 툴을 완성합니다.

👤 "이름변경 저장 기능 추가해볼까?"
